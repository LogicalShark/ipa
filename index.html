Instructions

Input: Enter some sample text to generate from. For phonetic generation, the text must be in English and will ignore punctuation.
File: Choose a pre-set text to generate from. Either provide your own input or choose a file.
First: The start of the output to initiate text creation. Must have the same number of characters as Order. Default: random.
Length: Enter the desired length of the output. Default: 1000 characters.
Order: Corresponds to the faithfulness to the original text. Higher order means faster computation and an output closer to the original text. Recommended range is 2 - 6 for short inputs. Default: 4.
Output: The output will be in IPA without stress or punctuation. You can read it yourself or use a tool like Amazon Polly to hear it spoken.

About

Uses the CMU Pronunciation Dictionary for phonetic data of English words and Firebase to store data.
